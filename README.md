## Project Overview
This project presents an interactive Power BI dashboard designed to analyze pizza sales performance. The objective is to turn raw transactional data into clear, actionable insights that support data-driven decision-making. The dashboard summarizes key business metrics, highlights sales trends, and provides product-level performance insights.

---

## Key Features

### KPI Summary
- Total Revenue: 817.9K  
- Total Orders: 21.4K  
- Total Pizzas Sold: 49.6K  
- Average Order Value: 16.8  

### Visual Components
- Sales trend over time  
- Revenue by pizza category  
- Revenue by pizza size  
- Top 10 best-selling pizzas  
- Sales distribution by hour  
- Interactive slicers for category and size
